Reference Value Explanation

The Treasury Reference Value is calculated by dividing the total TON reserve by the total NMX reserve. This represents the backing value of NMX tokens within the Nemex ecosystem treasury.

Important: This is the Treasury Reference Value — not a market price or token price. It reflects the treasury's internal reserve ratio and serves as a benchmark for the ecosystem's financial health.
